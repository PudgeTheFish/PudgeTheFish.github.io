Projects
  • AR Calculus Teaching Tool

    For an outreach program, my partner and I developed an augmented reality application to aid high school multivariable calculus students in visualizing shapes iterated over equations. I designed and developed the GUI, the skeleton that maps to the user's body, and the game environment. I also mapped the shape's position the user's hands so they can rotate and grow/shrink the model. The application was written in C# using Visual Studio and Unity, and it runs with an Xbox Kinect.

    You can download this application here.
  • Pets Tab - Chrome extension

    Pets Tab is a chrome extension that replaces the new tab page with aodrable images from popular instagram pets' recent posts. The user can select from cats, dogs, and birds to display, and their choice is stored for subsequent sessions. This application directly queries Instagram's API for media. I used HTML, CSS, JavaScript, and JQuery to create this extension.

    The extension is available in the Chrome store (linked here)
  • Message Alarm Clock - Android App

    This app is for the heavy sleepers and sleep deprived who need an extra push to wake up in the morning. Message Alarm Clock enables the user to set a message when they set their alarm time, and when the alarm goes off the next morning, the user must retype the message (which is displayed). This prevents the user absentmindedly shutting off their alarm and reminds them what they're waking up for. This app was developed in Java with Android Studio.
